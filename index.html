when to use id: The id attribute is used to point to a specific style declaration in a style sheet.
id="paragraph"

example:
#paragraph {
background-color: blue;
color: black;
padding: 40px;
text-align: center;
}

class : It is often used to point to a class name in a style sheet.

class example :
.attribute {
color: blue;
color: white;
border: 2px solid black;
margin: 20px;
padding: 20px;
}
